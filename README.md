# TheCatApi

For this project I would like to show us how to create a web page using some function with JavaScript, more specifically the Async/await function to control a promise

## Tech

In this project I use some technologies like:
- [TheApiCat] - The API that I used it 
- [JavaScript] - The main language that It used it
- [CSS] - CSS is used to define styles for my web pages
- [HTML] - HTML is a markup language for the web that defines the structure of web pages
- [bootstrap5] - Framework for creating responsive websites.
